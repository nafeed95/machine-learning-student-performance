# Student Performance Prediction

## Description
This project predicts student performance using machine learning algorithms.
Two classification models were implemented and compared.

## Algorithms Used
- Logistic Regression
- Decision Tree

## Dataset
Student performance dataset containing academic and demographic features.

## Results

| Metric    | Logistic Regression | Decision Tree |
|-----------|-------------------|---------------|
| Accuracy  | 86.08%            | 88.61%        |
| Precision | 95.65%            | 94.00%        |
| Recall    | 83.02%            | 88.68%        |
| F1-Score  | 88.89%            | 91.26%        |

## Conclusion
Decision Tree performed better overall with 88.61% accuracy and 91.26% F1-Score.
